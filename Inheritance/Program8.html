Inheritance

8. Write a Java program to create a class called Shape with methods called getPerimeter() and getArea(). Create a subclass called Circle that overrides the getPerimeter() and getArea() methods to calculate the area and perimeter of a circle.

Code


              // Base class Shape
class Shape {
    // Method to get the perimeter
    public double getPerimeter() {
        System.out.println("Calculating perimeter of a generic shape");
        return 0.0; // Default perimeter for a generic shape
    }
                                        
    // Method to get the area
    public double getArea() {
        System.out.println("Calculating area of a generic shape");
        return 0.0; // Default area for a generic shape
    }
}
                                        
// Subclass Circle extending Shape
class Circle extends Shape {
    // Attribute specific to Circle
    private double radius;
                                        
    // Constructor for Circle
    public Circle(double radius) {
        this.radius = radius;
    }
                                        
    // Override the getPerimeter method to calculate the perimeter of a circle
    
    public double getPerimeter() {
        double perimeter = 2 * Math.PI * radius;
        System.out.println("Calculating perimeter of a circle: " + perimeter);
        return perimeter;
    }
                                        
    // Override the getArea method to calculate the area of a circle
    
    public double getArea() {
        double area = Math.PI * radius * radius;
        System.out.println("Calculating area of a circle: " + area);
        return area;
    }
}
                                        
// Main class
public class ShapeExample {
    public static void main(String[] args) {
        // Create an instance of Shape
        Shape genericShape = new Shape();

        // Create an instance of Circle
        Circle myCircle = new Circle(5.0);

        // Demonstrate getPerimeter and getArea for both instances
        genericShape.getPerimeter();  // Output: Calculating perimeter of a generic shape
        genericShape.getArea();       // Output: Calculating area of a generic shape

        myCircle.getPerimeter();      // Output: Calculating perimeter of a circle: 31.41592653589793
        myCircle.getArea();           // Output: Calculating area of a circle: 78.53981633974483
    }
}

Output

Calculating area of a generic shape
