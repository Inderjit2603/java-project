Inheritance

1. Write a Java program to create a class called Animal with a method called makeSound(). Create a subclass called Cat that overrides the makeSound() method to bark.

Code


/ Base class Animal
class Animal {
    // Method to make a generic sound
    public void makeSound() {
        System.out.println("Animal makes a generic sound");
    }
}
                                        
// Subclass Cat extending Animal
class Cat extends Animal {
    // Override the makeSound method to bark
    
    public void makeSound() {
        System.out.println("Cat makes a 'Meow' sound");
    }
}
                                        
// Main class
public class AnimalExample {
    public static void main(String[] args) {
        // Create an instance of Animal
        Animal genericAnimal = new Animal();

        // Create an instance of Cat
        Cat myCat = new Cat();

        // Demonstrate makeSound for both instances
        genericAnimal.makeSound(); // Output: Animal makes a generic sound
        myCat.makeSound();         // Output: Cat makes a 'Meow' sound
    }
}

Output

Animal makes a generic sound 

Cat makes a 'Meow' sound
