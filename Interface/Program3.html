Interface

3. Write a Java program to create an interface Flyable with a method called fly_obj(). Create three classes Spacecraft, Airplane, and Helicopter that implement the Flyable interface. Implement the fly_obj() method for each of the three classes.

Code


// Interface
interface Flyable {
    void fly_obj();
}
                                        
// Spacecraft class implementing Flyable interface
class Spacecraft implements Flyable {
    
    public void fly_obj() {
        System.out.println("Spacecraft is flying in outer space");
    }
}
                                        
// Airplane class implementing Flyable interface
class Airplane implements Flyable {
    
    public void fly_obj() {
        System.out.println("Airplane is flying in the sky");
    }
}
                                        
// Helicopter class implementing Flyable interface
class Helicopter implements Flyable {
    
    public void fly_obj() {
        System.out.println("Helicopter is flying in the air");
    }
}
                                        
public class FlyableExample {
    public static void main(String[] args) {
        // Creating instances of different flying objects
        Spacecraft spacecraft = new Spacecraft();
        Airplane airplane = new Airplane();
        Helicopter helicopter = new Helicopter();
                                        
        // Calling the fly_obj method for each flying object
        spacecraft.fly_obj();
        airplane.fly_obj();
        helicopter.fly_obj();
    }
}

Output

Spacecraft is flying in outer space 

Airplane is flying in the sky 

Helicopter is flying in the air
